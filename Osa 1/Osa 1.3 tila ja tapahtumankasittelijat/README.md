# Osa 1.3
## Tila ja tapahtumankäsittelijät
Tässä osassa tutustutaan käsitteisiin komponentin tila sekä tapahtumankäsittelijä. Kansiosta löytyy esimerkkisovellus *counterEsimerkki*. Aloita tutustumalla esimerkkisovellukseen. Voit jälleen muokata koodia ja katsoa, mitä muutoksia ne aiheuttavat selaimessa. Varmista, että ymmärrät seuraavat käsitteet ja tiedät, miten niitä käytetään: *komponentin tila*, *useState-funktio*, *tapahtumankäsittelijä*.
